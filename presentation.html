Core Algorithms

ML + Statistical Models

🎯 3-Model Ensemble Credit Scorer

Weighted ensemble: XGBoost 50% + LightGBM 30% + Random Forest 20%

  • SMOTE oversampling (sampling_strategy=0.5)
  • Temporal train/test split (FY≤2017 / FY>2017)
  • Model Disagreement Signal: flags >0.30 spread as HIGH_UNCERTAINTY
  • SHAP TreeExplainer for feature attribution

Ref: Addo et al. 2018 — "Credit risk analysis using ML and DL models"

📊 Forensic Scoring Suite

  • Beneish M-Score — 8-ratio earnings manipulation detector (threshold >−2.22)
  • Altman Z-Score — 5-factor bankruptcy predictor (zones: Safe/Grey/Danger)
  • Piotroski F-Score — 9-component financial strength (0–9 scale)

🧬 Default DNA Fingerprinting

Matches borrower trajectory against 6 historical default archetypes: IL&FS, Kingfisher, DHFL, Café Coffee Day, Bhushan Steel, Jet Airways

DNA Match = 62% similar to Jet Airways

  • Velocity feature comparison (DSCR, D/E, margin slopes)
  • 30 companies × 12 fiscal years calibration set

🎲 Monte Carlo Stress Test

1,000 simulations across 5 RBI-calibrated macro shocks:

  • Repo Rate σ=1.5pp, Inflation σ=2.0pp, Revenue σ=15%
  • Commodity Price σ=20%, Customer Default U(0,0.30)
  • 3-year forward DSCR projection with compounding
  • Outputs: P10/P50/P90, Default Prob, Covenant Trigger

Implementation Details

Code & Data

🗂️ Dataset Generation — data_generator.py

  • 40 Indian companies (15 defaulted + 15 healthy + synthetics)
  • FY2009–FY2024 time series, 146 features per row
  • Calibrated from RBI NPA statistics with sector-specific rates
  • Deterioration factor (0→1) simulates gradual pre-default decline
  • Includes: P&L, Balance Sheet, Cash Flow, Governance, Auditor, Network, Satellite, GST, CEO sentiment

🧮 Feature Engineering — feature_engineering.py

  • 29 financial ratios: D/E, ICR, DSCR, Current Ratio, CCC, ROE, ROA, ln-transforms
  • Velocity features: YoY growth, ratio changes, DSCR acceleration, 3-yr slopes
  • Beneish M-Score: 8 components (DSRI, GMI, AQI, SGI, DEPI, SGAI, LVGI, TATA)
  • Altman Z-Score: 5 working-capital ratios → Safe/Grey/Danger zones
  • Piotroski F-Score: 9 binary signals → 0–9 financial strength

🏦 Credit Scoring — credit_scorer.py

  • Temporal split: train FY≤2017, test FY>2017 (no data leakage)
  • SMOTE when minority < 50% majority
  • XGBoost: 300 estimators, early stopping (30 rounds), AUC eval metric
  • Ensemble outputs: pd_ensemble, lending_decision, credit_limit_cr, risk_premium_pct
  • Decision logic: PD<0.35 → APPROVE, 0.35–0.60 → REVIEW, >0.60 → REJECT

🔗 Network Contagion — network_graph.py

  • Synthetic MCA21 data: 500 directors × 1000 companies × 2500 linkages
  • NetworkX bipartite graph → promoter subgraph extraction
  • Contagion Score = 0.4×NPA_ratio + 0.3×clustering + 0.3×size_factor
  • Risk levels: LOW (<0.25), MEDIUM (0.25–0.50), HIGH (>0.50)

📄 CAM Generator — cam_generator.py

  • 11-section DOCX with Vivriti-branded styling
  • Consistency validation rules (DSCR<1→REJECT, PD>40%→REJECT)
  • Auto risk badges, color-coded tables, SHAP explainability charts
