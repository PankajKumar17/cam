Yakṣarāja: Solution Overview

"Bridging the gap between raw financials and autonomous credit decisioning."

1. What is Yakṣarāja?

Yakṣarāja is an AI-Powered Credit Decisioning Engine designed to automate the end-to-end preparation of Comprehensive Credit Appraisal Memorandums (CAMs). Designed to reduce processing time from 5–7 days to under 3 minutes, it ingests raw financial datasets and alternative data points, runs them through rigorous statistical and agentic AI models, and synthesizes a production-ready DOCX report.

2. Key Objectives & Impact

  • Latency Reduction: Eliminates manual data entry and evaluation overhead (>99.9% faster).
  • Bias Removal: Employs entirely data-driven logic and dialectic LLM agents (Bull/Bear debate) to derive an objective credit sentiment.
  • Enhanced Accuracy: Explores 146 generated features (velocity arrays, financial forensical ratios) over large historical spans (FY2009–FY2024), detecting risks earlier than simple baseline models.

3. How it Works: The 10-Layer Engine

The system pipeline orchestrates multiple verification and scoring layers seamlessly:

  • Forensic Analysis: Evaluates earnings manipulation via Beneish M-Score and predictive insolvency using Altman Z-Score and Piotroski F-Score.
  • Credit Scoring: Uses a 3-model weighted ensemble (XGBoost, Random Forest, LightGBM) balanced with SMOTE for minority representation.
  • Alternative Data Verification:
    • MCA Network Graphs: Detects contagion across promoter links.
    • Satellite Feeds: Validates factory heatmaps and operational activity against disclosed capacities using Sentinel-2.
    • GST Cross-Check: Resolves invoice mismatches and tax divergence.
  • Stress Testing: Runs 1,000 Monte Carlo risk simulations simulating shocks (like RBI Rate hikes, commodity inflation).
  • Agentic Finalization: Orchestrates an LLM debate ending in an exhaustive Credit Appraisal Memo formatted intelligently using python-docx.

4. The Breakthrough Innovations

We utilized multiple innovations outside ordinary bounds. The top 3 being:

  1. Adversarial LLM Framework: Generative AI shouldn't just summarize — it must debate. Our Bull and Bear agents challenge data iteratively until a unified conclusion is securely established.
  2. Promoter Contagion Networks: Bipartite graph modeling isolates not just direct defaults, but hidden topological contagion.
  3. Satellite Truth Interlocks: Bypassing document forgery completely by comparing physical operational footprints generated externally.

5. Architecture & Stack

Running on Python 3.12, our system utilizes XGBoost and LightGBM for the core ensemble. LLM insights are powered by Anthropic's Claude and LangGraph frameworks, with data transformations handled by Pandas, Scikit-learn, and NetworkX.
